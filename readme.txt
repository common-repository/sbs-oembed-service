== Description ==

A oEmbed plugin that allows you to easy embed video content from [www.kijk.nl](http://www.kijk.nl) 

With this plugin you can embed all the video content from KIJK.NL just by copying a KIJK.NL video url into the content area of your WordPress post or page. See the screenshots for an example. 

On KIJK.NL you can find video content from the TV stations SBS6, Net5 and Veronica.

See [feeds.sbs.nl](http://feeds.sbs.nl) for more info


== Installation ==

This section describes how to install the plugin and get it working.

e.g.

1. Upload `sbs-oembed-service` to the `/wp-content/plugins/` directory
1. Activate the plugin through the 'Plugins' menu in WordPress
